开始使用
---
1. 将project文件夹导入到项目中
2. 修改config.properties中的数据库连接信息
3. 创建数据库，然后执行init.sql中的sql语句
4. 遵循[jfinalQ-gencode](http://git.oschina.net/uikoo9/jfinalQ-gencode)的表名规范新建表
5. 使用[jfinalQ-gencode](http://git.oschina.net/uikoo9/jfinalQ-gencode)生成代码
